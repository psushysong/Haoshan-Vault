# Morning Discovery — 5:00 AM

Copy this into Claude Code as a scheduled task.

**Folder:** Your vault root
**Repeats:** Every day at ~5:00
**Always allowed:** Bash, Read, WebFetch

## Prompt

```
Two jobs, do both:

1. EXPLORE: Read through my vault (Notes/, Projects/, Life/, journal.md, Inbox/). Find something I haven't touched in a while, or a connection between two notes I might not have noticed. Do a web search on a topic related to what you found — something adjacent, unexpected, that could give me a new angle. Don't repeat yourself in Inbox/. Do thorough research. Save the exploration note in Inbox/.

2. HARVEST (optional, skip if nothing is ready): Check Notes/ for any notes missing a Purpose line (Purpose: [[Project Name]]) at the bottom. If you can tell which project a note serves, add the Purpose link. Also check if any notes have matured enough to become an output — could they be turned into an article, a product feature, or a decision? If so, note this at the top of today's exploration note under a "Ready to harvest" heading. Most days there is nothing to harvest. That's fine. Explore is mandatory, harvest is not.
```
